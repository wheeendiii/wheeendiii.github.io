The building inventory csv file from uiuc ischool dataviz github and we also used it during class. The visualization takes in Congressional Districts as X and aggregated Square Footage values as Y. Instead of categorize square footage under different departments, the aggregated vaules regarding congressional district better shows distribution. Since we focus on each district, the default mark of this visualization is bar, showing a histogram-like graph type. However, the interaction function allows users change marks for other relational or aggregated informarion.
